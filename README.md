# Morse Code Translator 

یه مترجم ساده کد مورس با پشتیبانی از **فارسی** و **انگلیسی**.

---

## قابلیت‌ها :

- Encode (متن به مورس)
- Decode (مورس به متن)
- پشتیبانی از حروف فارسی و انگلیسی
- پشتیبانی از اعداد (۰-۹)
- پشتیبانی از علائم نگارشی (`,`, `!`, `?`, `.`, `@`)

---

## اجرا :

```
python3 MorseCodeTranslator.py
```
---
## نمونه خروجی encode :
```
1.encode
2.decode

```
